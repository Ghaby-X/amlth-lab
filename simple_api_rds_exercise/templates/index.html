SALES API

Welcome to my simple sales api; Explore the various api available

Endpoint Method Description
/top_customer_spends GET Returns the customers with the highest total spending.
/monthly_sales_report GET Provides a report of sales grouped by month.
/unordered_products GET Lists products that have never been ordered.
/average_order_by_country GET Displays average order values grouped by country.
/frequent_buyers GET Identifies customers with frequent purchases.
